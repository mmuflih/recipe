# Recipe

## Requirement
- golang version 1.7 or latest
- sql-migrate (https://github.com/rubenv/sql-migrate)

## How to Use
    git clone https://github.com/mmuflih/recipe.git
    cd recipe
    cp env.json.example env.json
    cp example.dbconfig.yml dbconfig.yml
    
## How to Run
    - sql-migrate up
    - go run main.go
